## Approach:
#hashmap 
tracker = {number: index of number}
- Done in single pass
```python
def twoSum(self, nums: List[int], target: int) -> List[int]:
	tracker = dict()
	for i, num in enumerate(nums):
		if (target-num) in tracker:
			return (tracker[target-num], i)
		else:
			tracker[num]=i
```
